# uni-bubble

## Development on Windows

### Install choco

### Install scoop

### Install sudo

```cmd
scoop install sudo
```

### Install virtualbox

```cmd
sudo choco install virtualbox
```

### Install vagrant

```cmd
sudo choco install vagrant
```

### Install vagrant cache

```cmd
vagrant plugin install vagrant-cachier
```

### Start vagrant

```cmd
vagrant up
```

### Build

```cmd
gradlew.bat
```
